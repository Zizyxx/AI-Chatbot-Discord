🤖 Discord AI Chatbot (Gemini Powered)Bot Discord berbasis Artificial Intelligence yang menggunakan API Google Gemini untuk memberikan pengalaman mengobrol yang natural, cerdas, dan interaktif. Bot ini dilengkapi dengan sistem memori permanen sehingga dapat mengingat konteks percakapan dan mengenali lawan bicaranya.✨ Fitur UtamaNatural Conversation: Menggunakan model gemini-2.5-flash dari Google untuk respon yang cepat dan natural layaknya manusia.Mention Only: Bot tidak akan spam di server; ia hanya merespon jika di-tag (@BotName).Persistent Memory: Dilengkapi dengan ingatan jangka panjang menggunakan sistem JSON lokal. Bot mengingat obrolan meskipun telah di-restart.User Recognition: Bot dapat membedakan dan mengingat siapa yang sedang berbicara dengannya berdasarkan display name di Discord.Cross-Channel Context: Ingatan bot bersifat global di dalam server, memungkinkan percakapan yang berkesinambungan lintas channel.🛠️ Persyaratan (Prerequisites)Sebelum menjalankan bot ini, pastikan Anda memiliki:Python 3.8 atau lebih baru terpasang di komputer Anda.Token Bot Discord (Dapatkan dari Discord Developer Portal).Google Gemini API Key (Dapatkan dari Google AI Studio).🚀 Cara InstalasiClone repository ini:git clone [https://github.com/USERNAME_ANDA/NAMA_REPO_ANDA.git](https://github.com/USERNAME_ANDA/NAMA_REPO_ANDA.git)
cd NAMA_REPO_ANDA
Install semua library yang dibutuhkan:pip install -r requirements.txt
Konfigurasi Environment Variables:Buat sebuah file bernama .env di direktori utama, lalu isi dengan format berikut:DISCORD_TOKEN=masukkan_token_discord_anda_disini
GEMINI_API_KEY=masukkan_api_key_gemini_anda_disini
Jalankan Bot:python bot.py
📝 Cara PenggunaanCukup undang bot ke server Discord Anda, lalu tag bot tersebut untuk mulai mengobrol!Contoh: @BotName Halo, nama saya Budi!Dibuat untuk tujuan pembelajaran dan pengembangan AI.
