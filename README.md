# AI-Chatbot-Discord

[![Python Version](https://img.shields.io/badge/python-3.9+-blue.svg)](https://www.python.org)
[![Discord.py](https://img.shields.io/badge/library-discord.py-7289DA.svg)](https://discordpy.readthedocs.io/en/stable/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

An advanced Discord bot powered by Google's Gemini Pro AI, designed to provide intelligent, contextual, and human-like responses within Discord servers. Built with performance and scalability in mind using asynchronous Python.

## ✨ Key Features
* **Powered by Gemini Pro:** Utilizes the latest Generative AI for high-quality reasoning and creative responses.
* **Contextual Conversations:** Remembers the flow of the conversation for a more natural interaction.
* **Slash Command Support:** Fully integrated with Discord's modern `/` commands for a better user experience.
* **Multi-Modal Support:** Capable of analyzing images and providing detailed descriptions or answers (if enabled).
* **Asynchronous Design:** Non-blocking operations ensuring the bot stays responsive even during heavy AI processing.

## 🛠️ Tech Stack
* **Language:** Python 3.9+
* **Framework:** [discord.py](https://github.com/Rapptz/discord.py)
* **AI Engine:** [Google Generative AI (Gemini)](https://ai.google.dev/)
* **Environment:** Dotenv for secure secret management.
